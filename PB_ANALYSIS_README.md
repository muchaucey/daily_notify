# PB值分析脚本使用指南

## 📋 概述

`pb_analysis.py` 是一个用于分析股票PB值历史趋势的Python脚本。该脚本主要针对**中国移动(601728)** 的历史数据进行分析，并生成详细的CSV报告。

## 🎯 功能特性

- ✅ 获取股票历史行情数据（开盘价、收盘价、最高价、最低价、成交量等）
- ✅ 计算多种技术指标（移动平均线MA5、MA20、MA50）
- ✅ 计算日涨跌幅
- ✅ 获取最新PB、PE、ROE等财务指标
- ✅ 生成详细的CSV分析报告
- ✅ 生成统计摘要文本报告
- ✅ 在命令行中输出统计汇总信息

## 📦 依赖环境

```
Python 3.7+
pandas >= 1.0.0
requests >= 2.25.0
```

## 📈 关键概念解释

### PB值（市净率）
- **定义**：股价 / 每股净资产(Book Value Per Share)
- **含义**：PB值越低，表示股票相对便宜
- **用途**：价值投资者常用的估值指标

### PE值（市盈率）
- **定义**：股价 / 每股收益(Earnings Per Share)
- **含义**：PE值越低，表示投资回报期越短
- **用途**：评估股票是否被高估或低估

### ROE（净资产收益率）
- **定义**：净利润 / 股东权益 × 100%
- **含义**：衡量公司利用股东资本的效率
- **用途**：判断公司盈利能力

## 🔧 配置选项

在脚本顶部可以修改以下配置：

```python
STOCK_CODE = "601728"        # 股票代码（默认：中国移动）
STOCK_NAME = "中国移动"       # 股票名称
ANALYSIS_DAYS = 500          # 分析天数（默认：500天）
OUTPUT_FILE = "pb_analysis_result.csv"  # 输出文件名
```

### 修改分析对象示例

若要分析其他股票，只需修改配置中的代码和名称：

```python
STOCK_CODE = "601919"   # 中远海控
STOCK_NAME = "中远海控"
```

常见股票代码：
- 601728: 中国移动
- 601919: 中远海控
- 601988: 中国银行
- 601398: 工商银行
- 000651: 格力电器（深圳）

## ⚠️ 常见问题

### Q1: 脚本连接失败
**A**: 这可能是由于：
- 网络连接问题：检查是否能访问外网
- API服务变更：远程API可能已更新，脚本会自动使用示例数据演示

脚本包含错误处理机制，会自动降级到示例数据模式，确保功能可用。

### Q2: 如何导入CSV到Excel？
**A**: 
1. 打开Excel
2. 选择 **文件** > **打开**
3. 选择 `pb_analysis_result.csv`
4. 选择合适的编码（建议UTF-8）

### Q3: 如何定期运行分析？
**A**: 

**Windows 任务计划程序**：
1. 打开"任务计划程序"
2. 创建新任务
3. 设置触发条件（如每天早上9点）
4. 设置操作为：`python pb_analysis.py`

**Linux/Mac Cron**：
```bash
0 9 * * * cd /path/to/project && python pb_analysis.py
```
